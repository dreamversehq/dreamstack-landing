## 🎯 Asset Quality Checklist

**Before Using Any Asset:**

### Visual Quality
- [ ] High resolution (no pixelation when zoomed)
- [ ] Proper color balance (not too dark/bright)
- [ ] Clean edges (no artifacts or blur)
- [ ] Consistent branding (colors match DreamVerse palette)

### Text Readability
- [ ] Font size large enough (minimum 24px for body text)
- [ ] High contrast between text and background
- [ ] No more than 3 font sizes per image
- [ ] Readable on mobile (test at 375px width)

### Brand Consistency
- [ ] Uses approved DreamVerse colors
- [ ] Professional typography (no Comic Sans!)
- [ ] Consistent with landing page design
- [ ] Logo placement consistent across assets

### Technical
- [ ] Correct dimensions for platform
- [ ] Optimized file size (fast loading)
- [ ] Proper file format (PNG for graphics, JPG for photos)
- [ ] Named correctly (dreamstack-cover.png, not IMG_1234.png)

## 🎨 Design Tools (Free Options)

If you need to edit your assets:

1. **Canva** (easiest)
   - Free tier has templates
   - Drag-and-drop interface
   - Export high-res PNG

2. **Figma** (most powerful)
   - Free for individual use
   - Professional design tool
   - Reusable components

3. **Photopea** (Photoshop alternative)
   - Free, browser-based
   - Photoshop-like interface
   - Works with PSD files

4. **Remove.bg** (background removal)
   - Free for basic use
   - Remove backgrounds from logos
   - Export transparent PNG

## 💡 Pro Tips

1. **Test on Real Devices:** View assets on your phone before launch
2. **Get Feedback:** Show to 2-3 people, ask "What stands out?"
3. **A/B Test Ideas:** Create 2 versions, pick the stronger one
4. **Brand Consistency:** Use exact hex codes, don't eyeball colors
5. **File Organization:** Name files clearly (dreamstack-twitter-banner.png)
